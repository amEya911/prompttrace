# Prompttrace Dashboard

> Visual analytics and ROI projections for your LLM context.

The Prompttrace Dashboard is a local-first Next.js application that visualizes the JSONL traces generated by your SDKs. It provides a high-level overview of your LLM spend, token hotspots, and potential savings.

---

## Features

- **ROI Projections** — Automatically projects current spend to monthly estimates at 10k/100k call scales.
- **Impact Simulations** — Live calculations of exactly how much money is being "wasted" on system prompt bloat.
- **Trace Explorer** — Deep-dive into individual message breakdowns, latency metrics, and model-specific costs.
- **Cache Hotspot** — Identifies repeated prompts that should be cached to save 100% of their cost.
- **Zero Cloud** — Reads directly from your local `.prompttrace/traces.jsonl` file.
- **High Performance** — Handles large trace files efficiently using backward streaming (no full file reads).

---

## Getting Started

### 1. Build the Dashboard
Ensure you are in the `apps/dashboard` directory:

```bash
npm install
```

### 2. Run the Server
```bash
npm run dev
```

### 3. Access the Analytics
Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Architecture

The dashboard functions as a **read-only consumer** of the `.prompttrace/traces.jsonl` pipeline.

1. **API Layer (`/api/traces`)**: Performs a high-performance backward chunk-read of the JSONL file to retrieve the latest 50 traces without loading the entire file into memory.
2. **Frontend Layer**: Renders the trace aggregate data, calculates simulations, and displays the "Severity" dots synchronized with the VSCode extension.

---

## Security & Storage Guarantees
- **Local First Guarantee:** Prompttrace runs exactly where your code runs. Traces stay inside `.prompttrace/traces.jsonl`. No code leaks.
- **Concurrency Safe:** The `.jsonl` append architecture natively handles extreme concurrency boundaries natively between Node.js and Python.

---

## 🔒 Privacy & Security

- **NO External Tracking**: This dashboard does not use Google Analytics or any external telemetry.
- **Local Data Only**: Your prompt data never leaves your machine. The "AI Analysis" features (optional) are powered by the extension provider you configure in VS Code.
